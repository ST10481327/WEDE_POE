# T.E.M Holdings Website

## Overview
T.E.M Holdings provides professional company registration, business consultancy, graphic design, and document fixing services in Johannesburg, South Africa. This website serves as a platform for clients to register their companies, submit inquiries, and interact with agents for support.

---

## Part 3 Updates
The following features were added in Part 3:

### **User Registration & Login**
- Clients can register with their personal and company details.
- Password validation ensures security (uppercase, number, special character, 9 characters long).
- Login system redirects users to a welcome page after successful authentication.
- Users can submit company issues and upload supporting documents for agents to review.

### **Service Selection & Agent Assignment**
- Dropdown menu with all company registration services and prices.
- Automatically assigns an agent to handle client requests.
- Contact options include WhatsApp, email, and online ticket submission.

### **Enhanced Pages**
- **About Page:** Company vision, mission, team images, video introduction.
- **Contact Page:** Form for inquiries with type selection (General, Support, Sales).
- **Gallery Page:** Display of design and documentation samples.
- **Location Page:** Embedded Google Maps with office location.

### **SEO Improvements**
- Optimized meta titles, descriptions, and keywords across all pages.
- Added `robots.txt` to guide search engine crawlers.
- Generated `sitemap.xml` for better indexing of pages.
- Added Open Graph and social media preview metadata.

### **Social Media Integration**
- Facebook, Instagram, Twitter (X), and LinkedIn links added to all pages.

### **Visual Enhancements**
- Unique background designs per page for improved UX/UI.
- Hero sections and styled forms for better aesthetics.
